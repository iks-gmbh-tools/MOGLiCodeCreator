The ExcelStandardModelProvider (since v1.5.0) can extent the function of the StandardModelProvider.

The ExcelStandardModelProvider can your class data from an excel file.

To use this functionality add the MetaInfo “useExtensionPlugin ExcelStandardModelProvider” on model level to your model file. This will cause the StandardModelProvider to use the ExcelStandardModelProvider to read the class data from the excel file defined in the _excel.properties of the ExcelStandardModelProvider.
